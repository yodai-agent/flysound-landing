# Flysound Landing Page

A modern, accessible landing page for Flysound hearing aids.

## Tech Stack
- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed to Vercel.
